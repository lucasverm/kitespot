# Kitespot
## Doel
* Kite trips sugereren op basis van wind en locatie in de buurt
* Verschillende bronnen van wind en locaties combineren
* Rekening houden met zonsopgang en zonsondergang

## Pagina's
* Spot info op basis van kaart
* User info aanpassen
* wind info op 1 pagina

## Kitespot data

* String naam
* int lat
* int long
* int windrichtingVan
* int windrichtingTot
* String beschrijving

## User data
* String voornaam
* String achternaam
* Thuis locatie
* Gewicht
* Beschikbare kites
